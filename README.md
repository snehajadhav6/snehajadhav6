A document Q&A chatbot using **HuggingFace embeddings + FAISS + Groq Llama 3.3 + LangChain**, deployed on Streamlit. Upload any PDF and ask questions — it finds the most relevant chunks and answers intelligently.

`Python` `LangChain` `FAISS` `Groq` `Streamlit`

---

#### Shnoor Import-Export Management System
Full-stack business dashboard for an import-export company — built with **React + FastAPI + PostgreSQL**. Features include shipment tracking, document OCR, HSN code lookup, revenue forecasting with linear regression (written from scratch in JS), risk analysis, and an AI chatbot.

`React` `FastAPI` `PostgreSQL` `Recharts` `Tailwind`

---

#### Spotify Music Popularity Predictor
Predicts whether a song will be popular using **Logistic Regression and Random Forest** on Spotify audio features. Trained, evaluated, and deployed with a clean Streamlit UI.

`Python` `scikit-learn` `Streamlit`

`Python` `scikit-learn` `TMDB API` `Streamlit`
